# Gustavo Savi — Frontend Developer Portfolio

Portfolio pessoal desenvolvido com React, TypeScript e Tailwind CSS, apresentando projetos práticos e habilidades técnicas voltadas ao desenvolvimento frontend.

🔗 **[gustavosavi.vercel.app](https://gustavosavi.vercel.app)**

---

## 🛠️ Tech Stack

| Tecnologia      | Uso                            |
| --------------- | ------------------------------ |
| React 19        | Componentização e lógica de UI |
| TypeScript      | Tipagem estática               |
| Tailwind CSS v4 | Estilização utilitária         |
| Framer Motion   | Animações e transições         |
| Vite            | Bundler e dev server           |

---

## 📁 Projetos em destaque

### 🎬 Movies App

Aplicação para busca e listagem de filmes consumindo a [TMDB API](https://www.themoviedb.org/documentation/api).

- Consumo de API REST com `fetch` e `async/await`
- Variáveis de ambiente com `import.meta.env`
- Roteamento com React Router

### ✅ To Do List React

Gerenciador de tarefas com operações CRUD completas.

- Gerenciamento de estado com `useState`
- Geração de IDs com `crypto.randomUUID()`
- Gadget com temperatura baseada na localização do usuário, além de dark mode

### 📍 Buscador de CEP

Ferramenta de consulta de endereço via [ViaCEP API](https://viacep.com.br/).

- Integração com API pública
- Tratamento de erros e estados de loading
- Validação de input

### 📋 CRUD Tarefas

Aplicação de gerenciamento de tarefas integrada a uma API externa (MockAPI).

- Operações GET, POST, PUT e DELETE
- Comunicação assíncrona com `fetch`
- Atualização reativa de estado

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/gutosavi/gustavo-portfolio.git

# Acesse a pasta
cd gustavo-portfolio

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse `http://localhost:5173` no navegador.
